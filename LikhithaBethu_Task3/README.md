# ATM Interface 💳

A simple ATM Interface application built using Java Swing. This project simulates basic ATM operations with a graphical user interface and demonstrates Java event handling, input validation, and GUI development.

## Features

✅ User Login Authentication

✅ Deposit Money

✅ Withdraw Money

✅ Transfer Money

✅ Check Account Balance

✅ Transaction History

✅ Logout Functionality

✅ Input Validation

## Technologies Used

- Java
- Java Swing
- Event Handling (ActionListener)
- OOP Concepts

## How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Open the project in VS Code or any Java IDE.

3. Compile the files:

```bash
javac LoginPage.java ATMInterface.java
```

4. Run the application:

```bash
java LoginPage
```
